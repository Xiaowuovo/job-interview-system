# 基于SpringBoot的就业面试辅助系统 - 设计文档

---

## 1. 系统概述

### 1.1 项目背景

随着就业竞争日益激烈，求职者需要系统化的面试准备工具。参考牛客网、LeetCode、拉勾网等主流平台，本系统聚焦于为求职者提供全方位的面试辅助服务。

### 1.2 系统定位

**核心价值**：打造一站式面试备考平台，涵盖知识学习、实战演练、智能评估、个性化推荐四大核心能力。

**目标用户**：
- 应届毕业生
- 求职转行者
- 在职提升者

### 1.3 核心竞争力

| 特性 | 说明 | 参考平台 |
|------|------|----------|
| 智能题库 | 分类、难度、标签多维度筛选 | LeetCode、牛客网 |
| AI模拟面试 | 真人对话式面试模拟 | HireVue、牛客网 |
| 知识图谱 | 结构化知识体系 | 拉勾教育 |
| 数据分析 | 个人能力雷达图、薄弱点分析 | BOSS直聘 |
| 社区互动 | 面经分享、问题讨论 | 牛客网、知乎 |

### 2.2 核心模块概览

#### 模块一：知识库管理模块 📚
**功能定位**：结构化面试知识体系

**核心功能**：
1. 知识图谱展示（技术栈、知识点关联）
2. 分类学习路径（Java、前端、算法、系统设计等）
3. 学习进度追踪
4. 知识点标注与笔记
5. 视频/文档/代码示例多媒体支持

**数据实体**：
- 知识分类（Category）
- 知识点（KnowledgePoint）
- 学习资源（Resource）
- 学习记录（StudyRecord）

---

#### 模块二：智能题库模块 💡
**功能定位**：覆盖全面的面试题库系统

**核心功能**：
1. **题目分类管理**
   - 按技术栈分类（Java、Python、前端、数据库等）
   - 按题型分类（选择题、编程题、简答题、系统设计题）
   - 按难度分级（简单、中等、困难）
   - 按公司标签（阿里、腾讯、字节等）

2. **智能刷题系统**
   - 随机练习模式
   - 专项突破模式
   - 错题本功能
   - 收藏夹功能
   - 每日一题推荐

3. **题目详情**
   - 题目描述
   - 多个参考答案
   - 答案解析
   - 相关知识点链接
   - 用户讨论区
   - 通过率统计

4. **在线编程环境**（针对算法题）
   - 代码编辑器
   - 多语言支持
   - 实时运行
   - 测试用例验证
   - 代码提交记录

**数据实体**：
- 题目（Question）
- 题目标签（QuestionTag）
- 用户答题记录（AnswerRecord）
- 代码提交（CodeSubmission）
- 错题本（WrongQuestionBook）

---

#### 模块三：AI模拟面试模块 🎤
**功能定位**：真实面试场景模拟

**核心功能**：
1. **面试类型选择**
   - 技术面试（算法、系统设计、项目经验）
   - 行为面试（STAR法则）
   - HR面试（职业规划、薪资谈判）

2. **智能面试官**
   - AI提问（基于岗位和简历）
   - 追问机制
   - 实时反馈
   - 语音/文字交互支持

3. **面试评估**
   - 回答完整性评分
   - 逻辑清晰度评分
   - 技术深度评分
   - 沟通表达评分
   - 综合评分与排名

4. **面试回放**
   - 完整对话记录
   - 关键问题高亮
   - 改进建议
   - 优秀回答参考

5. **真人模拟面试**（进阶功能）
   - 视频面试预约
   - 面试官匹配
   - 面试评价体系

**数据实体**：
- 面试会话（InterviewSession）
- 面试问题（InterviewQuestion）
- 面试评估（InterviewEvaluation）
- 面试反馈（InterviewFeedback）

---

#### 模块四：测评与分析模块 📊
**功能定位**：数据驱动的能力提升

**核心功能**：
1. **能力测评**
   - 入学测评（了解基础水平）
   - 阶段测评（检验学习成果）
   - 模拟笔试（真实考试环境）

2. **数据分析**
   - 个人能力雷达图
   - 知识点掌握度分析
   - 薄弱环节识别
   - 学习时长统计
   - 练习频率分析

3. **智能推荐**
   - 个性化学习路径
   - 题目推荐算法
   - 知识点推荐
   - 相似用户对比

4. **学习报告**
   - 周报/月报生成
   - 成长曲线展示
   - 目标达成度分析

**数据实体**：
- 测评记录（AssessmentRecord）
- 能力模型（AbilityModel）
- 学习统计（StudyStatistics）
- 推荐记录（RecommendationLog）

---

#### 模块五：用户与权限模块 👤
**功能定位**：用户体系与权限管理

**核心功能**：
1. **用户管理**
   - 注册/登录（手机、邮箱、第三方）
   - 个人信息管理
   - 简历管理
   - 目标岗位设置

2. **角色权限**
   - 学生（免费版、会员版）
   - 教师/面试官（内容创建、审核）
   - 管理员（系统管理）

3. **会员体系**
   - 免费用户（基础功能）
   - VIP会员（全部功能、无广告、优先服务）
   - 企业用户（团队协作、数据看板）

4. **积分体系**
   - 签到积分
   - 学习积分
   - 分享积分
   - 积分兑换

**数据实体**：
- 用户（User）
- 角色（Role）
- 权限（Permission）
- 会员记录（MembershipRecord）
- 积分记录（PointsRecord）

#### 模块五.5：教师端管理模块 👨‍🏫
**功能定位**：教师专属内容管理与教学辅助平台

**核心功能**：

1. **题库内容管理**
   - **题目创建与编辑**
     - 支持多种题型（选择题、简答题、编程题、系统设计题）
     - 富文本编辑器（支持代码高亮、图片、LaTeX公式）
     - 题目标签管理（技术栈、难度、公司、知识点）
     - 批量导入题目（Excel/JSON格式）

   - **题目审核流程**
     - 教师提交 → 管理员审核 → 发布上线
     - 审核状态：草稿、待审核、已通过、已拒绝
     - 审核意见反馈与修改建议

   - **题目质量管理**
     - 查看题目统计数据（答题人数、正确率、平均用时）
     - 学生反馈收集（错误报告、内容建议）
     - 题目优化建议（基于数据分析）

   - **参考答案与解析管理**
     - 多版本答案支持
     - 详细解析编写
     - 相关知识点关联
     - 扩展阅读资料链接

2. **知识库内容管理**
   - **知识点创建**
     - Markdown编辑器
     - 知识点分类与标签
     - 难度级别设置
     - 前置知识点关联

   - **教程创建**
     - 系列教程组织
     - 视频上传与管理
     - 代码示例库
     - 配套练习题

   - **内容素材库**
     - 图片素材管理
     - 代码模板库
     - 常用图表库
     - 引用资料库

3. **学生管理与数据分析**
   - **学生列表管理**
     - 查看教师课程下的学生
     - 学生学习进度查看
     - 学生能力评估报告
     - 学生学习轨迹分析

   - **作业与考试管理**
     - 创建练习任务
     - 设置截止时间
     - 自动批改（选择题、编程题）
     - 手动批改（简答题、系统设计题）
     - 成绩导出

   - **数据统计看板**
     - 学生活跃度统计
     - 题目完成率统计
     - 知识点掌握度分析
     - 薄弱知识点识别
     - 学习趋势图表

4. **课程与班级管理**
   - **课程创建**
     - 课程基本信息（名称、描述、封面）
     - 课程章节规划
     - 知识点与题目关联
     - 学习路径设计

   - **班级管理**
     - 创建班级
     - 学生邀请码生成
     - 学生名单管理
     - 班级公告发布

   - **教学计划**
     - 每周学习计划制定
     - 重点难点标注
     - 阶段性测验安排
     - 学习进度监控

5. **互动与反馈**
   - **学生问题答疑**
     - 学生提问列表
     - 在线回答问题
     - 常见问题整理（FAQ）
     - 问答历史记录

   - **评论与讨论管理**
     - 查看题目评论
     - 回复学生讨论
     - 精华评论置顶
     - 不当内容删除

   - **教学反馈收集**
     - 学生满意度调查
     - 课程评价查看
     - 改进建议收集
     - 教学效果评估

6. **个人中心（教师专属）**
   - **我的内容**
     - 我创建的题目（按状态筛选）
     - 我创建的知识点
     - 我的教程列表
     - 内容数据统计

   - **我的课程**
     - 授课课程列表
     - 班级管理入口
     - 课程数据概览

   - **收益与荣誉**
     - 内容贡献度积分
     - 学生好评率
     - 优秀教师认证
     - 平台推荐次数

   - **教师认证**
     - 实名认证
     - 专业领域认证
     - 教学资质上传
     - 个人简介编辑

**数据实体**：
- 教师信息（TeacherProfile）
- 课程（Course）
- 班级（Class）
- 作业任务（Assignment）
- 批改记录（GradingRecord）
- 教学统计（TeachingStatistics）
- 内容审核（ContentReview）

**教师端权限控制**：
```java
// 教师可访问的API端点
/api/teacher/questions/**      // 题目管理
/api/teacher/knowledge/**      // 知识点管理
/api/teacher/courses/**        // 课程管理
/api/teacher/classes/**        // 班级管理
/api/teacher/assignments/**    // 作业管理
/api/teacher/students/**       // 学生管理
/api/teacher/statistics/**     // 数据统计
/api/teacher/profile/**        // 个人中心
```

**教师端核心流程**：

1. **题目发布流程**
```
创建题目 → 填写详情 → 添加标签 → 预览 → 提交审核 → 审核通过 → 发布上线
```

2. **课程创建流程**
```
创建课程 → 规划章节 → 关联内容 → 设计路径 → 创建班级 → 邀请学生 → 发布作业 → 批改作业 → 数据分析
```

3. **学生管理流程**
```
查看学生列表 → 分析学习数据 → 识别薄弱点 → 针对性辅导 → 发布练习 → 跟踪进度 → 效果评估
```
