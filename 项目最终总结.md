## 🔧 技术架构

### 后端架构
```
Controller层（API接口）
     ↓
Service层（业务逻辑）
     ↓
Repository层（数据访问）
     ↓
Entity层（实体模型）
     ↓
Database（H2/MySQL）
```

### 前端架构
```
Views（页面组件）
     ↓
Router（路由管理）
     ↓
Axios（HTTP请求）
     ↓
Element UI（UI组件）
     ↓
ECharts（数据可视化）
```

## 📈 关键算法

### 1. 艾宾浩斯遗忘曲线
```java
// 计算复习时间
复习时间1 = 首次错误时间 + 1天
复习时间2 = 复习时间1 + 2天
复习时间3 = 复习时间2 + 4天
复习时间4 = 复习时间3 + 7天
```

### 2. 能力模型自动更新
```java
新分数 = (旧分数 × 0.7) + (本次正确率 × 100 × 0.3)
```

### 3. 连续签到算法
```java
遍历签到记录，从今天往前查找连续日期
```

### 4. 智能推荐算法
```java
推荐题目 = 薄弱分类题目 + 错题复习 + 未练习题目
```
