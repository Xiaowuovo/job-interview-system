## 🔄 核心业务流程

### 题目发布流程
```
创建 → 编辑 → 预览 → 提交审核 → 审核通过 → 发布
```

### 课程创建流程
```
基本信息 → 章节规划 → 内容添加 → 路径设计 → 发布
```

### 作业批改流程
```
发布作业 → 学生提交 → 自动批改 → 手动批改 →
评分评语 → 提交结果 → 通知学生
```

## 🎯 三端协同

```
学生端 ←→ 教师端 ←→ 管理端

学生学习行为
    ↓
生成学习数据
    ↓
    ├──→ 学生端：个人报告
    ├──→ 教师端：教学分析
    └──→ 管理端：平台统计
```
